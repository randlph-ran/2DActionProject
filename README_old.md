# 実習①の作業内容

**制作するもの**
 - 2Dアクションゲーム
 - MetroVania風のマップ探索型アクションゲーム
 - 実習①ではステージ1のみを作成
 - ステージ1のマップ構成はマップ3つ＝Scene3つ
 - マップ1と2にはザコ敵を複数体配置
 - PCのレベルアップあり　実習①では1レベル上がる体験を入れる　HPと攻撃力が上がる
 - ステージ奥のマップにいるボスを倒せばステージクリア
 - PCのHPが0になったらゲームオーバー

**仕様**
 - [ここにまとめ中](https://github.com/HiroshiKondo2026/2DActionProject/blob/main/%E4%BB%95%E6%A7%98/2D%E3%82%A2%E3%82%AF%E3%82%B7%E3%83%A7%E3%83%B3%E4%BB%95%E6%A7%98.xlsx)
 - 制作にはChatGPTを過分に使用中。ただし丸写しではなく、ソースコードへのコメント説明追加、および各メソッドやクラスの説明も入れてもらい、それをコメント入力して、あとで見返しても内容が理解できるようにしておく。＋わからない処理は都度確認しながら進行。
