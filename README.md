## プロジェクト概要 <a id="project-overview"></a>

| 項目 | 内容 |
|------|------|
| **ゲームタイトル** | Mutant&Vaiant |

| **一言で説明すると** | 2Dアクションゲーム |
| **ターゲット** | 同ジャンルを好む人たち |
| **制作の目的** | 2Dアクションゲームらしさを追加して、よりゲーム感を増す |
| **ゲームの目的** | ステージ奥にいるボスを倒す |
| **プレイヤー操作** | 例：WASD、矢印キー、ゲームパッド左スティックで移動、Z、左クリック、Aボタンで攻撃、スペースキー、右クリック、Bボタンでジャンプ、Vキー、Xボタンでアイテム使用、Bキー、Yボタンでガード |
| **勝利条件** | Stage2-3にいるボスの撃破 |
| **失敗条件** | HPが0になる |
| **特徴・工夫点** | プレイヤー操作性とアニメーションのリンク、敵をガードしつつ、倒して進むアクション性 |

## 開発ルール <a id="development-rules"></a>

- **Git運用**
  - 1〜2時間ごとに Commit
  - 1日 2〜3回 Push
  - 作業前に必ず **Pull** して最新を取得
  - Commit メッセージは内容がわかる日本語 or 英語（例：`プレイヤー移動を追加`）

- **Unityプロジェクト設定**
  - Unity 6（URP）を使用
  - 不要アセットは増やしすぎない（軽量管理）
  - `.gitignore` は Unity 公式テンプレートを使用

- **フォルダ構成（推奨）**
```
Assets/
├─ Animations
├─ Font
├─ Image
├─ Item
├─ Scripts/
├─ Prefabs/
├─ Scenes/
├─ Sprites/
├─ TextMeshPro
├─ TilePalette
├─ Tiles
└─ VFXPACK_IMPACT_WALLCOEUR_FreeVersion
```
